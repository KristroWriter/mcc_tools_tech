What is Help?

Help is content available to users from the application.

  • Worst case: A link to a PDF of your content.
  • Traditionally: A link to a help application or HTML frameset.

    If you're lucky, it's context sensitive.

  • Modernly: Opens a popup or panel with more information.
  • Go to hosted content.

What is a documentation portal?

Content hosted online

Often possible to get all content - all components, all versions, all languages, etc.

Sometimes behind a login

Sometimes connected to a knowledge base

Help vs Documentation Portal

This is a Venn diagram, not either/or.

Help is often installed with the product, but doesn't have to be.

Documentation Portals tend to be hosted, but may not be.

Both can be context-sensitive.
