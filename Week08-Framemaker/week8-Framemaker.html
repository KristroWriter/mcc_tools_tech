Unstructured vs Structured

There are two modes for FrameMaker

Unstructured (Authoring) - Define page layout and styles and use willy-nilly

(Preferably following your company's style guide)

Structured - Graphical representation of SGML/XML

Someone needs to define a structure, and you use elements instead of styles.

Middlesex Community College - Tools and Technologies for Tech Writers - Winter 2024

More on structured authoring

Overly simplified: Enforced style guide to ensure content is organized properly.

For example, you can define that when documenting a procedure, you need an introduction, a list of prerequisites, a lead in sentence, the procedure, and then results.

In unstructured authoring, you (or an editor) have to remember which styles to apply where. If you disagree with a style, you may "forget" to use them.

Structured authoring continued...

In structured authoring, the tooling enforces the styles and the structure.

A task topic must have a title.

The title in the task must be followed by an introduction.

The introduction can be followed by prerequisites. Prerequisites must be a list.

etc...

Books and files

There are three major types of files in FrameMaker

Book files - This file is a container for the content of your book.

Content files - The files that contain your content.

Generated files - Based on information in content files, create table of contents, list of things, index, etc.

Templates

FrameMaker is designed for print.

Templates include style catalogs, page layouts, color definitions, variables, cross-reference formats, etc.

You can import some or all of these items from any FrameMaker file into any other FrameMaker file.

Nifty features

Markers - Use to define index terms, markers for context sensitive help, and many other things.

Conditional text - You can hide comments from production builds, or mark text that should only appear for certain products or versions.

Cross-references - You can insert links to other parts of the document, including pre-defined formats

Variables - Specific phrases (with character tags) that can be reused and updated. For example, a version number or publication date.

Insert an image

  1. Use an anchor paragraph.
  2. Insert an Anchored Frame (Insert > Anchored Frame).
  3. Select the Anchored Frame.
  4. Insert a graphic (File > Import > File). Make sure you import by reference.
  5. Select the resolution for the image.

Anchored frames

FrameMaker is named after these "frames"

You can control where the frames go better than Word, so special objects go where you want them.

Frames can contain images, text, equations, etc.

Middlesex Community College - Tools and Technologies for Tech Writers - Winter 2024

Text frames

FrameMaker's answer to sub-file reuse.

You can re-use chapter files between books.

If you want to reuse a chunk of text, put it into a separate file in a named frame.

You can then reference that frame in other documents.

Generated files

Minimally create a Table of Contents (a list of specific headings with page numbers)

Often include an Index (an alphabetized list of index markers)

List of tables or list of figures.

For troubleshooting or clean up, you can get list of referenced files, etc.

You update the book to regenerate and update all references.
