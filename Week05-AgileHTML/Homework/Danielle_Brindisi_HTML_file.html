What is HTML?

HTML stands for HyperText Markup Language. It is a useful tool that helps web developers and the lay public alike create stunning webpages. Every page that you see on the internet has HTML written in the background. It is quite amazing that anyone can right click on the webpage and click the inspect button. You will then see your HTML code!

Here is a list of common terminology that you will use in HTML:

  • Tags-Used to hold the HTML element
  • Elements-Holds the content for HTML
  • Attributes-Used to describe the characteristic of an HTML element in detail

What is CSS?

CSS stands for Cascading Style Sheets. With CSS, you can easily design or style your website. This includes the layout, visual effects, and the background color of the webpage. There is a steep learning curve for CSS.

Here is a table that compares the pros and cons of CSS:

Pros of CSS Cons of CSS
Easy to maintain Browser Compatibility
Global Styling Steep Learning Curve
