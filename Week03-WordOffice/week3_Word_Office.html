Templates and Styles

They almost work...

A Word Template (.dot or .dotx) is a document that contains formatting options.

You need to keep them in C:\Users\<YourAccount>\Documents\Custom Office Templates

Often easier to create a new document based on a new template vs applying an existing template

(Applying an existing template assumes you are using the same style names)

Middlesex Community College - Tools and Technologies for Tech Writers - Winter 2022

Create new document

Make sure that your .dotx files are in your Custom Office Templates folder. (C:\Users\<YourUserName>\Documents\Custom Office Templates)

  1. Select File > New.

    Using CTRL+N doesn't get you to the template option.

  2. Select Personal next to Featured.

    Don't follow the Suggested Search for Personal.

  3. Select your template.

This creates a copy of template as a Word document. This should include boilerplate text and images.

Middlesex Community College - Tools and Technologies for Tech Writers - Winter 2022

Importing a template

This is for Word 2013, YMMV

  1. Select File > Options.
  2. Select Add-ins.
  3. Set Manage to Templates and click Go.
  4. Click Attach and find the .dotx or .dot file.
  5. Click OK.

You then need to re-apply styles as necessary. If you do this to a blank document, you will need to recreate the boilerplate content.

Using Styles

Stop using the quick formatting tools

The styles section of the ribbon or the panel are fine.

When formatting paragraphs, use paragraph styles with the ¶ mark.

For inline formating, use character styles with the a mark.

If you need to change formatting, go to the Style, right-click and select Modify.

Fixing Styles

Beware of Style Name + Formatting

The important thing about using Styles is that you should never, ever use any of the quick formatting options with them.

Of course, everyone else on the planet only ever uses the quick formatting options.

Things have changed over the years: https://www.techrepublic.com/article/three-ways-to-expose-format-inconsistencies-in-a-word-document/
